📅 Pagume Bot API

Advanced Ethiopian Calendar & Date Conversion Services (v1.0)

Welcome to the official developer documentation for Pagume Bot. Our API provides precise calendar conversions, age calculations, and real-time Ethiopian date data.

🔐 Authentication

All requests must include your API Key. We support Bearer authentication in the header (recommended) or query parameters.

Authorization: Bearer YOUR_API_KEY

Or via URL:

?key=YOUR_API_KEY

🚀 Core Endpoints

GET /v1/convert

Convert between Gregorian and Ethiopian calendars.

GET /v1/convert?date=18/04/2026&to=ethiopian&key=YOUR_KEY

GET /v1/today

Fetch current date in both calendar systems.

GET /v1/today?key=YOUR_KEY

GET /v1/age

Calculate precise age including Ethiopian months.

GET /v1/age?birth_date=10/08/2018&calendar=ethiopian&key=YOUR_KEY
