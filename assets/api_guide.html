📅 Pagume Bot API

Official Developer Documentation (v1.0)

Welcome to the Pagume Bot Developer API. This API provides high-precision Ethiopian calendar services, including date conversion, current date retrieval, and age calculation.

🔐 Authentication

All requests must include your API Key. We support Bearer authentication in the header (recommended) or query parameters.

Authorization: Bearer YOUR_API_KEY

Or via URL parameter:

GET /api/endpoint?key=YOUR_API_KEY

🚀 Core Endpoints

GET /api/convert

Convert dates between Gregorian and Ethiopian calendars.

ParameterDescription
dateRequired. The date in DD/MM/YYYY format.
toRequired. Target calendar: ethiopian or gregorian.
GET /api/convert?date=21/04/2026&to=ethiopian&key=YOUR_KEY

GET /api/today

Fetch current date in both calendar systems simultaneously.

GET /api/today?key=YOUR_KEY

GET /api/age

Calculate precise age including Ethiopian months and days.

ParameterDescription
birth_dateRequired. Birth date in DD/MM/YYYY format.
calendarOptional. Input calendar type (gregorian or ethiopian).
GET /api/age?birth_date=10/08/2018&calendar=ethiopian&key=YOUR_KEY

📦 Response Format

All responses are returned as JSON with the following structure:

{
  "success": true,
  "data": { ... },
  "meta": { "timestamp": "2026-04-21T15:00:00" }
}
